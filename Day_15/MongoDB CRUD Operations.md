# Day 15 – MongoDB CRUD Operations

## Work Summary
Today I studied MongoDB CRUD operations and learned how to interact with a MongoDB database using different queries. I explored the four main operations: Create, Read, Update, and Delete. 

I practiced inserting documents into a collection, retrieving data using query filters, updating existing records, and deleting documents when needed. I also understood how MongoDB stores data in a flexible JSON-like format and how collections manage multiple documents.

This practice helped me understand how MongoDB works as a NoSQL database and how it can be used in backend applications.
