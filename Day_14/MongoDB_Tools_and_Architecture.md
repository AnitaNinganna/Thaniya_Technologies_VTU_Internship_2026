# Day 13 - Exploring MongoDB Tools and Architecture

## What I worked on

### Work Summary
Today I continued learning MongoDB and explored how MongoDB stores data internally. I learned that MongoDB stores data in BSON (Binary JSON) format which helps in efficient storage and retrieval.

I studied the MongoDB architecture which consists of Databases, Collections, and Documents. Collections contain multiple documents and each document stores related information in JSON-like format.

I also learned about MongoDB tools such as MongoDB Compass, which is a graphical interface that helps developers visualize databases, explore collections, and insert or edit documents easily.

Additionally, I explored MongoDB Atlas which is the cloud-hosted version of MongoDB. It allows developers to create scalable databases in the cloud and access them from anywhere.

Finally, I studied real-world use cases of MongoDB such as social media applications, e-commerce platforms, and content management systems.
