# Day 18 – MongoDB Atlas Learning

## Work Summary
Today I focused on learning MongoDB Atlas, which is a cloud-based database service provided by MongoDB. I explored how to create and manage a cluster, connect the database to applications, and use the MongoDB Atlas dashboard.

I learned how to set up a free tier cluster, configure database users, and whitelist IP addresses for secure access. I also practiced connecting MongoDB Atlas to a local environment using a connection string.

Additionally, I explored how data is stored and managed in the cloud, and understood the advantages of using MongoDB Atlas over local databases, such as scalability, availability, and ease of management.
